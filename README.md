# EduAlign
EduAlign is a submission and review support platform designed specifically for **absestos abatement training** providers focusing on **Level 1: Foundational Awareness** programs.<br>
It supports current WorkSafeBC approval process by structuring:
- how program course materials are,
- how provider self-assessments are completed,
- and how competency alignment is validated with the **Asbestos Abatement Training: Curriculum Mapping Template.**
## Platform Overview
- Reduces back-and-forth between reviewers and applicants.
- Improves the accuracy of competency mapping.
- Creates a clear, traceable record of validation decisions.

EduAlign AI helps asbestos training providers submit complete, well-aligned applications while enabling reviewers to validate competency coverage, instructional methods, and assessment approaches more efficiently and consistently.
